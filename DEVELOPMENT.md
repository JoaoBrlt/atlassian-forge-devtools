# Development

## Requirements

- [pnpm](https://pnpm.io/installation)
- [Node 24](https://nodejs.org/en/download)

## Install

- Install the dependencies:

```bash
pnpm install
```

## Start

- Start the extension:

```bash
pnpm run dev
pnpm run dev:firefox
```

## Format

- Check the format:

```bash
pnpm run format
```

- Format the code:

```bash
pnpm run format:fix
```

## Lint

- Lint the code:

```bash
pnpm run lint
```

- Fix the lint issues:

```bash
pnpm run lint:fix
```

## Validate

- Validate the types:

```bash
pnpm run typecheck
```

## Test

- Run the tests:

```bash
pnpm run test
```

## Build

- Build the extension:

```bash
pnpm run build
pnpm run build:firefox
```

## Package

- Package the extension:

```bash
pnpm run zip
pnpm run zip:firefox
```
